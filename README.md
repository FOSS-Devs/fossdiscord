# FossiumBot
Welcome to the GitHub page of FossiumBot!
The bot is written in C# using .NET and [DSharpPlus](https://dsharpplus.github.io)

## Features
The bot has a lot of features, including but not limited to:
- Moderation
  - Warning system
  - Kick, ban, softban
  - Purge
  - Autodelete
- Utils
  - Avatar, userinfo, serverinfo
- Fun
  - Dog, cat
  - Wikipedia
